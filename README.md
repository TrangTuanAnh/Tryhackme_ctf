## Thêm box mới

1. Copy thư mục `writeups/_template/` → `writeups/<TênBox>/`
2. Tạo thư mục `writeups/<TênBox>/images/` để chứa ảnh
3. Điền nội dung vào `README.md` theo khung có sẵn
4. Thêm 1 dòng vào bảng trên
